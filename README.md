# excel-trans
本项目是一款专为跨境物流与 IT 行业设计的智能化 Excel 翻译解决方案。依托 智谱 AI (GLM-4) 大模型，本工具打破了传统翻译软件破坏表格格式的痛点，实现了在保留原始复杂布局（包括合并单元格、多工作表、富文本样式）的前提下，进行高精准度的跨语言转换。
技术栈： Python | Streamlit | ZhipuAI (GLM-4) | Openpyxl

技术亮点：

解析逻辑： 采用 Openpyxl 深入底层 XML 逻辑，精准定位合并单元格锚点，避免重复翻译。

富文本处理： 实现了对 CellRichText 对象的深度迭代，支持单元格内单字级别的样式保留。

智能 Prompt 工程： 通过动态注入源语种/目标语种上下文，结合物流领域知识库，提升 AI 在混装文本中的表现。

异步体验： 在 Streamlit 框架下实现流式进度反馈与内存级文件流导出，无临时文件残留。
